Publish into Google Cloud Pub/Sub

  1. Open a page (Valid / Duplicate / Invalid).
  2. Generate JSON → Copy JSON.
  3. Go to Console → Pub/Sub → Topics → YOUR TOPIC → Publish message.
  4. Paste JSON in Message body → Publish.

Tip: For valid JSON, you can add attribute content_type=application/json (optional).

Message Rules

  • Use a unique event_id for each new event (dedup key).
  • order_id stays the same for the same order.
  • OrderCreated includes order info and an items array.
